---

## 서비스 정의

### 핵심 컨셉
트위터/포스타입에 흩어진 **이메레스, 트레틀, 페어틀**을 한 곳에서 검색하고, **포토샵 없이 웹에서 바로 페어틀을 편집**할 수 있는 플랫폼

### 용어 정리

| 용어 | 설명 |
|------|------|
| **자캐** | 자작 캐릭터 (Original Character) |
| **자캐러** | 자캐를 만들고 활동하는 사람 |
| **페어** | 두 캐릭터의 관계/커플링 |
| **페어틀** | 두 캐릭터를 함께 넣을 수 있는 템플릿 이미지 |
| **이메레스** | 이미지 레퍼런스, 캐릭터 프로필/정보 템플릿 |
| **트레틀** | 트레이싱 틀, 그림 연습용 밑그림 템플릿 |
| **틀** | 위 모든 템플릿의 통칭 |

---

## 타겟 유저

### 핵심 타겟: 트위터 자캐 커뮤러
- **연령대**: 10대 후반 ~ 30대 초반
- **성별**: 여성 비율 높음 (약 80%)
- **특성**:
  - 모바일 중심 사용
  - 페어 문화 활발
  - 창작/팬활동 적극적
  - 포토샵 등 전문 툴 접근성 낮음

### 확장 타겟: TRPG 유저, 창작러
- 캐릭터 시트 제작 니즈
- 관계도/세계관 정리 니즈
- 협업 창작 니즈

### 유저 페르소나

#### 페르소나 1: 민지 (22세, 대학생)
- 트위터에서 자캐 활동 2년차
- 포토샵 없어서 페어틀 사용 못함
- 페어 상대방과 함께 틀 채우고 싶음
- 모바일로 주로 활동

#### 페르소나 2: 수아 (28세, 직장인)
- 이메레스/페어틀 제작하는 틀 작가
- 자신의 틀이 널리 쓰이길 원함
- 부수입 창출 관심
- 무단 도용 걱정

## 핵심 가치 제안 (Value Proposition)

| 기존 문제 | Pairy의 해결책 |
|----------|---------------|
| 틀이 트위터/포스타입에 흩어져 있음 | 한 곳에서 통합 검색 |
| 포토샵 없으면 페어틀 못 씀 | 웹 에디터로 누구나 편집 |
| 혼자 만들어야 함 | 실시간 협업 기능 |
| 무단 도용 걱정 | 워터마크 + 보호 서비스 |
| 틀 작가 수익화 어려움 | 마켓플레이스 + 후원 |

## 경쟁 우위 & 차별화

### 직접 경쟁자 없음
- **틀 통합 검색**: 자캐 틀 전문 아카이브 서비스 없음
- **페어틀 웹 에디터**: 자캐 특화 협업 에디터 없음
- **실시간 협업**: Figma/Canva는 자캐 문화 미특화

### vs 경쟁 서비스

| 서비스 | 영역 | Pairy 차별점 |
|--------|------|-------------|
| 크레페 | 커미션 중개 | 틀 편집 기능 없음 |
| 포스타입 | 콘텐츠 판매 | 틀 검색/편집 불가 |
| 캔바 | 디자인 도구 | 자캐 특화 아님, 협업 약함 |
| 픽크루 | 캐릭터 메이커 | 틀과 다른 영역, 수익화 없음 |

## 브랜딩

### 이름 의미
**Pairy** = Pair(쌍) + y = 페어(커플) + 요정(Fairy)의 중의적 표현

### 브랜드 키워드
- 마법 (Magic)
- 함께 (Together)
- 창작 (Creation)
- 연결 (Connection)

### 톤 & 매너
- **친근하고 귀여운** 말투
- **창작자를 존중**하는 태도
- **커뮤니티 중심** 분위기
- 전문적이되 **딱딱하지 않은** 느낌

### 로고 컨셉
- 🧚 요정 날개 + 하트
- 💕 두 캐릭터가 연결된 모습
- ✨ P 글자에 반짝이 효과
- 🎀 리본으로 묶인 두 프레임

## MVP 집중 전략

### 핵심 원칙: "2인 페어틀 하나에 올인"
모든 틀 유형을 지원하기보다, 가장 인기 있는 **2인 페어틀**에 집중하여 에디터 사용성을 극대화

### MVP 범위

#### ✅ MVP에서 하는 것
- 2인 페어틀 에디터 완성도
- 실시간 2인 협업
- 원클릭 테마컬러 변경
- 자동 저장
- PC/모바일 이원화

#### ❌ MVP에서 안 하는 것
- 3인 이상 틀
- 복잡한 레이어 편집
- AI 노이즈 보호
- 음성/화상 채팅
- API 연동
