# 🧚 Pairy - 페이지 레이아웃 (Page Layouts)

## 페이지 목록

### MVP 페이지 (P0-P1)
1. 랜딩 페이지 `/`
2. 틀 아카이브 `/templates`
3. 틀 상세 `/templates/[id]`
4. 에디터 `/editor/[id]`
5. 협업 에디터 `/collab/[sessionId]`
6. 마이페이지 `/my`
7. 로그인 (모달)
8. 검색 결과 `/search`

### 확장 페이지 (P2-P3)
- 작가 프로필 `/creator/[id]`
- 작가 대시보드 `/creator/dashboard`
- 프리미엄 안내 `/premium`
- 설정 `/settings`
- 공지사항 `/notice`

### 3.2 틀 카드 컴포넌트

```jsx
<article className="group cursor-pointer">
  {/* 썸네일 */}
  <div className="relative aspect-[4/3] rounded-xl overflow-hidden">
    <Image
      src={template.thumbnail}
      alt={template.title}
      fill
      className="object-cover group-hover:scale-105 transition-transform"
    />
    {/* 호버 오버레이 */}
    <div className="absolute inset-0 bg-black/40 opacity-0 group-hover:opacity-100 transition-opacity flex items-center justify-center">
      <Button>에디터로 열기</Button>
    </div>
    {/* 가격 뱃지 */}
    {template.price > 0 && (
      <span className="absolute top-2 right-2 bg-accent-500 text-xs px-2 py-1 rounded-full">
        ₩{template.price}
      </span>
    )}
  </div>

  {/* 정보 */}
  <div className="mt-3 space-y-1">
    <h3 className="font-medium truncate">{template.title}</h3>
    <p className="text-sm text-gray-500">@{template.creator}</p>
    <div className="flex items-center gap-3 text-sm text-gray-400">
      <span className="flex items-center gap-1">
        <Heart size={14} /> {template.likes}
      </span>
      <span className="flex items-center gap-1">
        <Download size={14} /> {template.uses}
      </span>
    </div>
  </div>
</article>
```
